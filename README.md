# Vibe Cluster Insight Hub

한국 인프라 클러스터 데이터 통합 시각화 및 분석 플랫폼

---

## 🏗️ 프로젝트 개요

**Vibe Cluster Insight Hub**는 국가철도공단, 국토교통부, 한국공항공사, 해양수산부 등 다양한 공공기관의 인프라 데이터를 통합하여, 지도 기반 시각화와 정량적 분석, 클러스터링, 성장성 예측, 네트워크 분석, 추천 시스템 등 다양한 기능을 제공하는 대시보드형 웹 애플리케이션입니다.

- **주요 목적**
  - 전국 인프라(철도, 도로, 공항, 항만) 현황 및 계획 데이터 통합
  - 산업단지 및 인프라 입지의 접근성, 성장성, 네트워크 등 다각적 분석
  - 정책 수립, 투자유치, 입지 선정 등 실무 활용 가능한 인사이트 제공

---

## 🚩 주요 기능

- **지도 기반 시각화**: 네이버 지도 API를 활용한 인프라 위치 및 클러스터 표시
- **클러스터 분석**: 지역별 인프라 밀도, 산업단지 집적도 분석
- **성장성 예측**: 교통, 고용, 투자, 인력 등 다양한 지표 기반 성장성 점수 산정
- **네트워크 분석**: 산업단지 및 기업 간 협업 가능성, 공급망 네트워크 시각화
- **시나리오 시뮬레이션**: 정책 변화, 신규 인프라 개통 등 가상 시나리오 반영 분석
- **입지 추천 시스템**: 업종/기업명/KSIC 코드 기반 최적 산업단지 추천
- **PDF 리포트 생성**: 분석 결과를 PDF로 저장 및 공유

## ✅ 접근성 종합 점수 산정 방식

### 1. 고속도로 접근성 (가중치 0.25)

- **지표:** 산업단지 ↔ 최근접 고속도로 IC까지의 거리
- **정규화 방식:**
  - 0km → 100점, 먼 거리일수록 점수 감소
  - 거리 값에 대해 Min-Max Scaling (선형 정규화) 적용
- **사용 데이터:**
  - 국토교통부\_도로 공사정보\_20241119.csv (도로명, IC명, 위도, 경도)
  - 충청북도\_산업단지현황\_20240729.csv (산업단지명, 위도, 경도)
- **계산 방식:**
  - 각 산업단지와 모든 IC 간의 Haversine 거리 계산 후 가장 가까운 IC 선택
  - 해당 거리값을 기반으로 정규화 점수 부여

### 2. 철도 접근성 (가중치 0.25)

- **지표:** 산업단지 ↔ 최근접 철도역까지의 거리
- **보정 요소:**
  - 고속철도역: +10점 보정
  - 화물역/환승역 등: +5점 보정
- **사용 데이터:**
  - 국가철도공단\_철도건설현황\_20240619.csv
  - 국가철도공단\_고속철도 사업계획\_20240719.csv
  - 국토교통부\_철도산업 사업현황\_20240730.csv
  - (노선명, 역명, 역 종류, 위도, 경도)
- **계산 방식:**
  - 산업단지 ↔ 가장 가까운 철도역 거리 계산
  - 해당 역의 종류에 따라 점수 보정
  - 최종 거리 점수에 선형 정규화 적용

### 3. 공항 접근성 (가중치 0.15)

- **지표:** 산업단지 ↔ 최근접 공항 거리 + 인프라 수준
- **보정 요소:**
  - 여객/화물 항공편 수, 주차장, 계류장 면적 등 고려
  - 인프라 많은 공항은 +10~20점 가산
- **사용 데이터:**
  - 한국공항공사\_공항 위치정보\_20240801.csv (공항명, 위도, 경도)
  - 한국공항공사\_공항 계류장 및 주차장 정보\_20240719.csv (계류장수, 여객터미널수, 화물터미널수)
  - 한국공항공사\_공항별 교통정보\_20240726.csv
- **계산 방식:**
  - 산업단지 ↔ 공항 거리 계산 (Haversine)
  - 공항 인프라 수준에 따라 점수 가산
  - 선형 정규화로 0~100 범위 점수 산정

### 4. 항만 접근성 (가중치 0.10)

- **지표:** 산업단지 ↔ 최근접 항만 거리
- **보정 요소:**
  - TEU 처리능력 (컨테이너 수용량)
  - 항만 건설 공사 건수 및 규모
  - 항만 수심 및 접안능력 (가능한 경우 추가)
- **사용 데이터:**
  - 해양수산부\_항만건설사업 공정현황정보\_20121231.csv
  - 해양수산부\_항만건설사업 사후평가\_20210101.csv
  - (항만명, TEU, 공사 수, 진척률, 위도, 경도)
- **계산 방식:**
  - 산업단지 ↔ 가장 가까운 항만까지 거리 계산
  - TEU + 공사 규모를 0~100 점수로 정규화
  - 거리 정규화 점수와 가중 평균하여 항만 접근성 점수 생성

### 🎯 종합 접근성 점수 계산 공식

```
종합접근성점수 =
  (고속도로 점수 × 0.25) +
  (철도 점수 × 0.25) +
  (공항 점수 × 0.15) +
  (항만 점수 × 0.10)
```

- 모든 점수는 0~100 범위로 정규화된 상태
- 보정 점수는 항목 내에서 ±10~20점 이내에서 가중 적용
- 단일 항목의 극단값이 종합점수에 과도한 영향을 주지 않도록 제한 적용

#### 📊 예시: 오창제2 산업단지 (산업단지 위경도 기반 분석 결과)

| -------- | -------- | ------------ | --------------------------- |
| 고속도로 | 1.2      | 96.5점       | 가장 가까운 IC 기준         |
| 철도     | 3.5      | 91.2점       | 일반역 + 보정               |
| 공항     | 8.0      | 85.3점       | 청주공항 + 여객 인프라 반영 |
| 항만     | 140.0    | 76.9점       | 평택항 기준, TEU/공사 반영  |
| 총합     | -        | 57.05점      | 가중 평균 계산 결과         |

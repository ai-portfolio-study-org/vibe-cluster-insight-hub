# spec.md

## 🔍 웹앱명: 충북 산업 클러스터 AI 분석 플랫폼 (VIBE-CLUSTER.AI)

---

## 🎯 웹앱의 목표

충청북도 산업단지 및 신산업 클러스터 관련 공공데이터를 기반으로, 지역 내 산업단지 입지 추천, 교통/물류 최적화, 기업 협업 가능성 시각화, 성장성 예측 기능을 통합한 **AI 기반 산업 입지 전략 도구**를 제공한다.
이를 통해 투자유치, 입지선정, 지자체 정책 수립에 **직접 활용 가능한 정량·정성 분석 결과**를 리포트로 제공한다.

---

## ⚙️ 메인 기능

### 1. 산업단지 입지 추천 AI

- 업종명 또는 KSIC 코드 입력 시 최적의 산업단지를 추천
- 추천 근거 시각화 (입지 집적도, 교통성, 고용지표, 유사기업 수)

### 2. 물류 및 교통 접근성 분석

- 청주 교통 빅데이터 연동
- 산업단지별 물류 유입·이동 경로 시뮬레이션
- 혼잡도, 이동시간, 유턴비율 시각화

### 3. 지역별 산업 유망도 예측

- 읍면동 단위 고용, 투자, 인력, 교통지표 분석
- 향후 성장가능성 스코어 예측 및 지도 시각화
- 가상 산업단지 확장 시 성장 점수 변동 시뮬레이션 포함

### 4. 기업 유형 클러스터링

- 입주 기업의 업종, 생산품, 고용형태 기반 자동 클러스터 분류
- 특화 산업군 탐색 및 유사군 비교 기능

### 5. 기업 협업 네트워크 시각화

- 산업단지 내 공급망 가능성, 기술 협업 유망 파트너 추천
- 협력망 GNN 기반 시각화 제공
- 정책 변화 시 예상 협업 구조 변화 탐색 기능 포함

## 🔁 유저 흐름

1. 사용자 로그인
2. 기업명 or 업종 코드 입력 → 입지 추천
3. 주변 기업 네트워크 확인 → 협업 파트너 시각화
4. 해당 지역 고용/교통 데이터 확인 → 입지 유망성 점수 확인
5. 가상의 산업단지 조성 시나리오 추가 → 입지 변화 시뮬레이션 확인
6. 결과를 PDF 리포트로 저장 → 전략 제안서 첨부 자료 생성

✅ 접근성 종합 점수 산정 방식
📌 1. 고속도로 접근성 (가중치 0.25)
지표: 산업단지 ↔ 최근접 고속도로 IC까지의 거리

정규화 방식:

0km → 100점, 먼 거리일수록 점수 감소

거리 값에 대해 Min-Max Scaling (선형 정규화) 적용

사용 데이터:

📂 국토교통부_도로 공사정보_20241119.csv

열: 도로명, IC명, 위도, 경도

📂 충청북도_산업단지현황_20240729.csv

열: 산업단지명, 위도, 경도

계산 방식:

각 산업단지와 모든 IC 간의 Haversine 거리 계산 후 가장 가까운 IC 선택

해당 거리값을 기반으로 정규화 점수 부여

📌 2. 철도 접근성 (가중치 0.25)
지표: 산업단지 ↔ 최근접 철도역까지의 거리

보정 요소:

고속철도역: +10점 보정

화물역/환승역 등: +5점 보정

사용 데이터:

📂 국가철도공단_철도건설현황_20240619.csv

📂 국가철도공단_고속철도 사업계획_20240719.csv

📂 국토교통부_철도산업 사업현황_20240730.csv

열: 노선명, 역명, 역 종류, 위도, 경도

계산 방식:

산업단지 ↔ 가장 가까운 철도역 거리 계산

해당 역의 종류에 따라 점수 보정

최종 거리 점수에 선형 정규화 적용

📌 3. 공항 접근성 (가중치 0.15)
지표: 산업단지 ↔ 최근접 공항 거리 + 인프라 수준

보정 요소:

여객/화물 항공편 수, 주차장, 계류장 면적 등 고려

인프라 많은 공항은 +10~20점 가산

사용 데이터:

📂 한국공항공사_공항 위치정보_20240801.csv

열: 공항명, 위도, 경도

📂 한국공항공사_공항 계류장 및 주차장 정보_20240719.csv

열: 계류장수, 여객터미널수, 화물터미널수

📂 한국공항공사_공항별 교통정보_20240726.csv

계산 방식:

산업단지 ↔ 공항 거리 계산 (Haversine)

공항 인프라 수준에 따라 점수 가산

선형 정규화로 0~100 범위 점수 산정

📌 4. 항만 접근성 (가중치 0.10)
지표: 산업단지 ↔ 최근접 항만 거리

보정 요소:

TEU 처리능력 (컨테이너 수용량)

항만 건설 공사 건수 및 규모

항만 수심 및 접안능력 (가능한 경우 추가)

사용 데이터:

📂 해양수산부_항만건설사업 공정현황정보_20121231.csv

📂 해양수산부_항만건설사업 사후평가_20210101.csv

열: 항만명, TEU, 공사 수, 진척률, 위도, 경도

계산 방식:

산업단지 ↔ 가장 가까운 항만까지 거리 계산

TEU + 공사 규모를 0~100 점수로 정규화

거리 정규화 점수와 가중 평균하여 항만 접근성 점수 생성

🎯 종합 접근성 점수 계산 공식
text
복사
편집
종합접근성점수 = 
(고속도로 점수 × 0.25) + 
(철도 점수 × 0.25) + 
(공항 점수 × 0.15) + 
(항만 점수 × 0.10)
모든 점수는 0~100 범위로 정규화된 상태

보정 점수는 항목 내에서 ±10~20점 이내에서 가중 적용

단일 항목의 극단값이 종합점수에 과도한 영향을 주지 않도록 제한 적용

철도	3.5	91.2점	일반역 + 보정

항만	140.0	76.9점	평택항 기준, TEU/공사 반영
총합	-	57.05점	가중 평균 계산 결과
